Why Chuti?

Everything you need,
nothing you don't

Built specifically for small organisations that need a reliable, no-fuss leave tracking system without ongoing cloud costs.

🔒

Fully Offline

Your data never leaves your building. All records are stored in a local SQLite database that you fully control.

📡

LAN Sharing

Install on one PC and every colleague on the same Wi-Fi can access it from their browser. No extra setup required.

📂

Choose Your Data Folder

Pick exactly where your database, backups, and uploaded documents are saved. Full control, no surprises.

🔄

Auto Daily Backups

The database is automatically backed up every time the application starts. Up to 30 rolling backups are kept.

📊

Leave Analytics

Real-time dashboard with leave balance tracking, late deductions, sandwich rule enforcement, and holiday management.

📎

Document Attachments

Attach medical certificates and other files to leave records. All uploads are stored in your chosen data folder.

🖱

One-Click Install

Download the installer, run it, choose a data folder on first launch — and you're ready. No command line, no Node.js needed.

Get Started

Up and running
in 60 seconds

No developer experience required. If you can install any Windows program, you can install Chuti.

1

Download the installer

Click the Download button and save Chuti-Setup.exe to your computer.

2

Run Chuti Setup.exe

Double-click the installer. Windows may show a SmartScreen prompt — click "More info → Run anyway" (the app is unsigned but safe).

3

Choose your data folder

On first launch, a dialog will ask you to choose a folder for your database and files. Pick any folder you like (e.g. Documents\ChutiData).

4

Share the LAN URL with your team

A popup will show the Network URL (e.g. http://192.168.1.x:3000). Share this with colleagues on the same Wi-Fi.
