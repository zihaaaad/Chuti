Features

Everything you need,
nothing you don't

Built for small organisations that need a reliable system without ongoing complexity or cost.

Fully Offline

Your data never leaves your building. All records are stored in a local SQLite database that you fully control and can back up at any time.

LAN Sharing

Install on one machine and every colleague on the same Wi-Fi can access it from their browser — no additional setup required.

Choose Your Data Folder

On first launch, pick exactly where your database, backups, and uploaded files are saved. Full control, no surprises, no hidden locations.

Auto Daily Backups

The database is backed up automatically every time the application starts. Up to 30 rolling backups are kept so you can always restore.

Leave Analytics

Real-time dashboard with leave balance tracking, late deductions, sandwich rule enforcement, holiday management, and summary reports.

Document Attachments

Attach medical certificates and supporting documents to leave records. All uploads are stored safely in your chosen data folder.

Setup

Running in
under a minute

No developer experience required. If you can install any Windows program, you can install Chuti.

01

Download the installer

Click the download button below and save Chuti-Setup.exe to your computer.

02

Run the setup

Double-click the installer. Windows SmartScreen may appear — click "More info" then "Run anyway." The app is safe and open-source.

03

Choose a data folder

On first launch, a dialog asks you to choose a folder for your database and files. Pick anywhere — for example Documents\ChutiData.

04

Share the network URL

A popup shows your network address (e.g. http://192.168.1.x:3000). Share it with colleagues on the same Wi-Fi so they can log in from their browsers.
